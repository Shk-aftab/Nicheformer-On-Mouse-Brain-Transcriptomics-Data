# Evaluation Contract

## Purpose
Ensure that centralized and federated results are comparable
and reproducible.

---

## Required Evaluations
- Global test set evaluation
- Per-client test set evaluation

---

## Required Metrics
- Accuracy
- Macro-F1
- Per-client accuracy

---

## Required Plots
- Confusion matrix
- Accuracy per client
- Training curves

---

## Output Files
- eval_summary.json
- plots/*.png

---

## Milestone 4 Extension
Fairness metrics, robustness analysis, and spatial consistency
can be added here.
